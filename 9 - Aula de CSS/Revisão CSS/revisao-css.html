Revisão de CSS

Identificadores (id) e classes (class)

O valor do atributo id deve ser único para cada elemento em uma página.

O valor do atributo class pode se repetir para mais de um elmento na mesma página.

Um elemento pode ter id e classe ao mesmo tempo.

Um elemento pode ter mais de uma classe.

As diferentes classes de um elemento (valor atributo class) devem ser separadas com espaço.

Formas de usar CSS

  1. Inline: CSS em linha. Utiliza o atributo style.
  2. Interno: Utiliza a tag style para definir blocos de declarações CSS dentro do documento HTML. Geralmente é colocado no head da página.
  3. Externo: Utiliza um arquivo .css para definir os estilos. Qualquer documento .html pode importar esse arquivo. Usa-se a tag link para fazer a imporatação.

Hierarquia do funcionamento do CSS

As definições de estilo mais próximas aos elementos são as que prevalecem.

CSS inline (em linha) está mais próximo dos elementos do que CSS interno e externo.

Entre CSS interno e externo, o que vai estar mais próximo dos elementos depende da ordem de criação deles dentro do documento .html.
