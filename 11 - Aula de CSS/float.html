Usando a propriedade clear.

A propriedade clear define o que deve acontecer com o elemento que vem depois de um elemento "flutuante" (elemento que o valor left ou right para a propriedade float).

Há cinco valores possíveis, left, right, both, none e inherit

  • Left: faz com que o elemento não seja ajustado no espaço que foi liberado pelo elemento flutuante caso ele esteja flutuando para a esquerda (float: left).
  • Right: faz com que o elemento não seja ajustado no espaço que foi liberado pelo elemento flutuante caso ele esteja flutuando para a direita (float: left).
  • Both: faz com que o elemento não seja ajustado no espaço que foi liberado pelo elemento flutuante, independente da direção que foi usada.
  • None: faz com que o elemento seja ajustado sempre.
  • Inherit: faz com que o elemento herde a definição do elemento pai.
