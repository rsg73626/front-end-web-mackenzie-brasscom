Posicionamento com CSS (propriedade position).

Static

Esse elemento está posicionado com o método estático (valor static para a propriedade position no CSS).

Posicionamentos relativos

Relative

Elementos posicionados relativamente têm sua posição alterada pelo valor das propriedade top (topo), bottom (parte de baixo), left (lado esquerda) e right (lado direito). O posicionamento do elemento será calculado a partir da sua posição original levando em conta o valor dessas quatro propriedades.

Fixed

Elementos posicionados relativamente à tela, ou seja, usando o valor fixed para a propriedade position, têm sua posição alterada pelo valor das propriedade top (topo), bottom (parte de baixo), left (lado esquerdo) e right (lado direito).

O posicionamento dos elementos fixos será calculado a partir do valor dessas quatro propriedades com relação ao tamanho da tela do computador.

Elementos posicionados fixamente não se movem mesmo que a página seja escrolada.

Absolute

Elementos que usam o valor absolute para a propriedade position podem ser posicionados relativamente ao elemento.

Usa-se as propriedade top, bottom, left e right para posicionar os elementos.

Caso o elemento esteja dentro do body ele será posicionado com relação ao corpo da página, e quando a mesma escrolar, o elemento também irá se mover.

O elemento pai ao qual o posicionamento será relativo deve usar o posicionamento relative.

Sticky

É usado para posicionar o elemento de acordo com o scroll da página.

Em outras palavras, é possível deixar o elemento fixo na tela quando ele atingir determinada posição, definiada com as propriedade top, bottom, right e left.
