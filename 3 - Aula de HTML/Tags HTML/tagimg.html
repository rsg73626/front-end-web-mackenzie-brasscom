Tag de imagem

A tag de imagem, <img>, é utilizada para inserir imagens de uma página web. Ela possui algumas configurações importantes que devem ser feitas.

Toda tag <img> precisa de um valor para o atributo src, que deve ser a URL com o caminho da imagem a ser exibida. Esse caminho pode ser uma URL de uma imagem na internet, ou o caminho de uma imagem que está em alguma pasta do mesmo computador na qual a página está salva. Por organização o ideal é que a imagem a ser usada esteja na mesma pasta da página web (arquivo .html), ou em uma subpasta que agrupa as imagens do site (por exemplo, "imagens").

Também é importante definir um valor para o atributo alt (texto alternativo), que será exibido quando não for possível exibir a imagem. Também é importante para a acessibilidade, pois quem não enxerga consegue ter acesso ao mesmo conteúdo da imagem. Caso a imagem seja apenas decorativa, ou seja, para implementar o design da página, pode-ser usar um valor vazio ("").

Outro atributo que pode ser interessante usar é o title, que definir um texto com algum conteúdo sobre a imagem que será exibido quando usuário passar o cursor em cima da imagem.

É possível definir a largura e a altura da imagem diretamente pelo HTML (através dos atributos width e height). Mas o ideal é usar o atributo style e definir o tamanho da imagem com as propriedades do CSS.
