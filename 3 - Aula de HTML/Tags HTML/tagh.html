Tags de título

Tags de título são usadas para definir o título das diferentes seções de uma página. Há diferentes níveis, de 1 a 6, sendo primeiro o nível mais importante, e o último o nível menos importante.

Tags de título são escritas no format <hn>, onde n deve ser substituído pelo nível do título, ou seja, um valor entre 1 e 6.

Idealmente, deve haver somente um título de nível 1 em cada página. Deve-se seguir uma hierarquia de títulos, ou seja, somente usar um h2 depois de h1, somente usar um h3 depois de um h2, e assim por diante.

A seguir, exemplos de uso de cada um dos nível de título:

  • Isso é um título de nível 1.

  • Isso é um título de nível 2.

  • Isso é um título de nível 3.

  • Isso é um título de nível 4.

  • Isso é um título de nível 5.
  • Isso é um título de nível 6.
