Tags de lista

Tags de lista, <ul>, <ol> e <li>, são utilizadas para que possamos exibir listas em nossas páginas. É possível criar listas com marcadores não numéricos, e listas com marcadores numéricos.

As listas com marcadores não numéricos são chamadas listas desordenadas, e a tag utilizada é a tag <ul> (abreviação de unordered list).

As listas com marcadores numéricos são chamadas listas ordenadas, e a tag utilizada é a tag <ol> (abreviação de ordered list).

Os itens de ambos os tipos de listas são especificadas com a tag <li> (abreviação de list item). Não é correto usar essa tag fora de uma <ul> ou <ol>. É possível criar listas aninhadas (uma dentro da outra).

A seguir, exemplos de utilização das tags de lista:

Exemplo de lista desordenada (<ul>)

  • Item 1
  • Item 2
  • Item 3

Exemplo de lista ordenada (<ol>)

  1. Item 1
  2. Item 2
  3. Item 3

Exemplo de listas aninhadas (uma dentro da outra)

  1. Item 1
  2. Item 2
  3. Item 3
    1. Item 3.1
    2. Item 3.2
    3. Item 3.3
      • Item 3.3.1
      • Item 3.3.2
      • Item 3.3.3
