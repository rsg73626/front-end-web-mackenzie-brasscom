Variáveis JS

Em JavaScript há três palavras reservadas para a criação de variáveis (containers para armazenamento de dados):

  • var
  • let
  • const

var x let x const

var

  • O escopo das variáveis é global.
  • Variáveis criadas dentro de um bloco podem ser acessadas fora desse bloco.
  • Variáveis podem ser redeclaradas.
  • Variáveis podem ser usadas antes de serem declaradas.

let

  • O escopo das variáveis é em bloco.
  • Variáveis criadas dentro de um bloco não podem ser acessadas fora desse bloco.
  • Variáveis não podem ser redeclaradas no mesmo escopo.
  • Variáveis não podem ser usadas antes de serem declaradas.

const

  • Possui as mesmas características do uso do let, com duas adições.
  • Ao ser declarado, as constantes precisam de um valor (não é permitido declarar a constande e DEPOIS atributior valor).
  • Não é permitido mudar o valor das constantes.

Regra guida

Para definir se deve usar var/let ou const pergunte-se se o valor da variável vai mudar ao longo do programa. Se sim, use var ou let. Se não use const (ou seja, crie uma constantes).
