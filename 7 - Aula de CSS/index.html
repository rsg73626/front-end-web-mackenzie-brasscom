Background (fundo dos elementos)

A seguir, algumas maneiras de se estilizar o fundos dos elementos via CSS:

  • Background-color
  • Background-image
  • Background-image padrão
  • Background-image sem repetição
  • Background-image repetindo somente em x (horizontal)
  • Background-image repetindo somente em y (vertical)
  • Background-image arredondando (preenche todo o fundo)
  • Background-image com espaçamento (preenche todo o fundo)

Posição da imagem de fundo (background-position)

  • left top
  • left center
  • left bottom
  • center center
