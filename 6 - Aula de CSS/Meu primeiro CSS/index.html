Exemplos de CSS

Uso do CSS inline (em linha)

Consiste em definir declarações CSS como valor do atributo style nas tags de abertura dos elementos HTML. Pode ser usada para qualquer elemento da minha página.

Uso do CSS interno (dentro do arquivo .html)

Consiste em utilizar a tag style e definir os códigos CSS (composto de seletor e bloco de declarações) como conteúdo dessa tag. A tag style pode ser colocada em qualquer lugar do documento, mas geralmente é colocada dentro da tag head.

Uso do CSS externo (em um arquivo .css separado)

Consiste em definir um novo arquivo .css, independente de qualquer outra arquivo, que pode ser importado para as nossas diferentes páginas HTML.

Seletores

Seletores apontam para os elementos que devem receber o estilo especificado no bloco de declarações CSS. Há diferentes tipos de seletores:

  • Seletor por elemento: usa como seletor o nome de um elemento.
  • Seletor por identificador: usa como seletor o valor do atributo id de algum elemento do documento.
  • Seletor por classe: usa como seletor o valor do atributo class de algum elemento do documento.

Esses seletores podem ser usados em conjunto para múltiplas seleções (cada seletor deve ser separado por vírgula).
